# ViewCartDemo
A sample demo project for showing the view cart screen using android architecture components with MVVM design patterns
## Prerequisites
- Android SDK v27
- Latest Android Build Tools v27.1.1
- Android Support Repository
- Retrofit2
- Picasso
## Getting Started
This demo project uses the Gradle build system.

Download the ViewCartDemo by cloning this repository or downloading an archived snapshot. (See the options at the top of the page.)
In Android Studio, create a new project and choose the "Import Project" option.
Select the ViewCartDemo directory that you downloaded with this repository.
If prompted for a gradle configuration, accept the default settings. Alternatively use the "gradlew build" command to build the project directly.
